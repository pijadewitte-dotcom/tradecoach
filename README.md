# Tradecoach

Een eenvoudige Nederlandstalige macro trade coach. Je vult een aandeel, koers, trenddata, volatiliteit en risicoprofiel in. De app maakt daarna een handelsplan met:

- koopzone en breakout-alternatief
- stopniveau en positiegrootte
- verkoopniveaus
- timing-checklist
- trend-, setup- en risicoscore

De stijl is geinspireerd door een macro/trend/risk-first manier van denken. Dit is educatief en geen financieel advies.

## Gebruik

Open `index.html` in je browser. Er is geen installatie of backend nodig.
